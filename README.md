# Zephron

Zephron is a browser-based personal assistant dashboard with a local Python server, task tracking, note capture, quick prompts, and optional voice features.

## Run It

```bash
python main.py
```

The app serves the `frontend` folder on `http://127.0.0.1:8000/` by default and opens a browser automatically.

## What It Does

- Chat-style assistant interface
- Add, complete, and remove tasks
- Save and clear notes
- Schedule reminders with desktop alerts when supported
- Export reminders to calendar files or share them by email
- Quick prompts for common actions
- Local storage persistence in the browser
- Voice input and speech output when supported by the browser
